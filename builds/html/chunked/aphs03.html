H.3. Using MoniCA

Once MoniCA starts up, you can select the observatory that you wish to monitor. The Navigator drop-down menu is where you will find the different display pages that have been defined. The pages are displayed in a hierarchical format for easier navigation. However, it might take a while to become familiar with the pages are available.

Figure H.1. The navigation menu in MoniCA.


Each page is made up of one or more components called Panels. For example, there are panels for “tables of current values” and “time series graphs”. Tabs enable you to control what each panel displays. You can also add or remove panels from the current display through the Setup drop-down menu, in this manner new display pages can be created. If you create a page that you want to keep, you can save that setup to your local computer through the Setup menu.
